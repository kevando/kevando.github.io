# Local Development


### Local Development


```
bundle exec jekyll serve --livereload
```

This website is built with Github Pages. 





```
gem install bundler
gem install jekyll
bundle install
bundle exec jekyll serve
```

```
bundle add webrick
gem uninstall eventmachine
bundle install
bundle exec jekyll clean
```
